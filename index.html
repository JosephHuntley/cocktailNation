About Us

Cocktail Nation is a project constructed to demonstrate the use of APIs on a website. The API that this website is getting its cocktail recipes from is TheCocktailAPI.

Cocktail Nation represents a make-believe liquor store that hosts a cocktail recipe finder to encourage customers to buy more supplies from their store.
